Cada proyecto se trabaja de una forma distinta

Nos tomamos el tiempo de analizar cada proyecto meticulosamente para proponer la mejor propuesta acorde a las necesidades de nuestros clientes. Ningún proyecto es igual ni se trabajan de la misma manera. Estamos en constante contacto para resolver cualquier punto y hacer el trabajo de la manera mas eficiente posible.

Nuestro proceso de trabajo

Este es nuestro proceso una vez que se acepta el trabajo, somos muy detallistas para ofrecer el mejor resultado posible.

Se hará la presentación hacia el cliente detallando cada punto sobre como se va a trabajar sobre el proyecto, incluyendo tiempo y presupuesto estimado del trabajo.

Se firmará el contrato entre ambas partes. Una vez hecho esto se iniciará el desarrollo del proyecto.

Cuando el proyecto esté a punto de terminarse, el cliente deberá abonar la parte faltante de la liquidación para la entrega del trabajo.

Si el cliente acepta, se cobrarán las comisiones iniciales sumando el 60% anticipado del proyecto principal.

Habrá comunicación constante con el cliente a medida que se va desarrollando el proyecto. Habrá rondas de ajustes hasta que el cliente quede conforme.

Una vez entregado el trabajo con la aprobación del cliente, habrá 30 días de soporte y Hosting bonificado y quedamos atentos a posibles errores futuros durante ese lapso de tiempo.
